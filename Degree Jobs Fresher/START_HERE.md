## ⚠️ Important Notes

### Remember:
1. **Personal Access Token** is needed for git push
   - Get from: https://github.com/settings/tokens
   - Use as password (not your GitHub password)
   - Keep safe (can regenerate anytime)

2. **Replace YOUR_USERNAME** in commands
   - Example: If your username is `john-smith`
   - Replace in: `https://github.com/john-smith/degree-jobs-fresher-bot.git`

3. **CHAT_ID is important**
   - Make sure it's: `-1003877235401` (negative number)
   - Copy exactly as is

4. **Secrets are case-sensitive**
   - `BOT_TOKEN` (all caps)
   - `CHAT_ID` (all caps)
